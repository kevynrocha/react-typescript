## Começando

Para obter uma cópia local e funcional, siga essas etapas

### Instalação

1. Clone o repositório
```sh
git clone https://github.com/kevynrocha/react-typescript
```
2. Instale as dependências
```sh
yarn
```
## Utilização

1. Inicie o projeto
```sh
yarn start
```
